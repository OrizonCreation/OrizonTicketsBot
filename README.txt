English Version

Welcome to the ReadMe for the guided installation of The Orizon Tickets Bot.

Open Install_Dependencies.bat to install the required dependencies.

Open main.py and paste your bot token on line 15.

Once the bot is online, you can configure the ticket panel using the /ticketset command.

Versione Italiana 

Benvenuto nel file ReadMe per l’installazione guidata di The Orizon Tickets Bot.

Apri Install_Dependencies.bat per installare le dipendenze necessarie.

Apri main.py e incolla il token del tuo bot alla riga 15.

Quando il bot sarà online, potrai configurare il pannello dei ticket con il comando /ticketset.
